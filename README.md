### Micro Frontend Demo
A micro frontend setup demonstrating Module Federation with React. A container app loads independent micro frontends at runtime.

Architecture
* Container app on port 3000 loads three remotes: Header on 3001, Content on 3002, Footer on 3003
* Module Federation shares React and React DOM as singletons to avoid duplicate runtime copies
* Each micro frontend runs independently, enabling isolated development and testing

Tech
* React 18, Webpack 5, Module Federation, CRACO
